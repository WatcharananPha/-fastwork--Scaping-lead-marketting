- สร้าง Dashboard และรายงานเชิงวิเคราะห์ โดยใช้เครื่องมือ BI (Power BI, Tableau, Looker Studio ฯลฯ)
- สื่อสารกับฝ่ายธุรกิจเพื่อเข้าใจความต้องการ และนำเสนอ Insight ที่มีคุณค่า
- เขียน SQL เพื่อดึงข้อมูลและประมวลผลตามเงื่อนไข
- ตรวจสอบความถูกต้องของข้อมูล และทำ Data Cleaning/Transformation
- ช่วยวางแผนกลยุทธ์เชิงข้อมูล (Data-driven decision making)
- ทำรายงานประจำสัปดาห์ / เดือน เพื่อให้ผู้บริหารใช้ในการตัดสินใจ
Requirement
- วุฒิปริญญาตรีขึ้นไป ในสาขาวิทยาการคอมพิวเตอร์, สถิติ, บริหารธุรกิจ, หรือสาขาที่เกี่ยวข้อง
- มีประสบการณ์ด้าน Data Analysis / Business Intelligence อย่างน้อย 3 ปี
- มีทักษะ SQL (เช่น SELECT, JOIN, GROUP BY)
- ใช้เครื่องมือ BI ได้ เช่น Power BI, Tableau, Looker Studio
- เข้าใจแนวคิด Data Warehouse, ETL, Data Modeling พื้นฐาน
- มีทักษะการนำเสนอข้อมูลและการสื่อสารกับทีมธุรกิจ
- หากมีทักษะ Python (Pandas/Matplotlib) หรือ Excel ระดับสูง จะได้รับการพิจารณาเป็นพิเศษ

 สรุปภาพ: Data Analyst vs Data Scientist vs Business Analyst
——————————
เหมาะกับ:
 • นักเรียนสาย Data / IT
 • คนอยากเปลี่ยนสายงานมา Data
 • องค์กรที่กำลังวางแผนสร้างทีมวิเคราะห์ข้อมูล
⸻
 Data Analyst
หน้าที่: วิเคราะห์ข้อมูลที่มีอยู่ เพื่อช่วยให้ธุรกิจตัดสินใจดีขึ้น
ทักษะ: SQL, Data Visualization, Reporting, Basic Stats
เครื่องมือ: Excel, Tableau, Power BI, Python
 เงินเดือน: ₹5L – ₹9L /ปี
 เหมาะกับผู้เริ่มต้นในสายงานข้อมูล
⸻
 Data Scientist
หน้าที่: สร้างโมเดล Machine Learning วิเคราะห์ข้อมูลเชิงลึก เพื่อขับเคลื่อนกลยุทธ์
ทักษะ: Math, Programming, ML, Deep Learning, Stats
เครื่องมือ: Python, R, TensorFlow, PyTorch, Hadoop
 เงินเดือน: ₹7L – ₹12L /ปี
 เหมาะกับสายลึก สายวิจัยข้อมูลและ AI
⸻
 Business Analyst
หน้าที่: เข้าใจปัญหาธุรกิจ และใช้ข้อมูลในการเสนอแนวทางแก้ไข
ทักษะ: Communication, Process Modeling, Stakeholder Management
เครื่องมือ: MS Office, BI Tools
 เงินเดือน: ₹4.8L – ₹7L /ปี
 เหมาะกับคนชอบพูดคุย วิเคราะห์เชิงธุรกิจ
⸻
 สรุปสั้นที่สุด
 • Data Analyst = วิเคราะห์ข้อมูลพื้นฐาน 
 • Data Scientist = วิเคราะห์เชิงลึก ใช้ AI 
 • Business Analyst = เข้าใจธุรกิจ แก้ปัญหาด้วยข้อมูล 
 “เริ่มจากจุดเล็กๆ แล้วเติบโตเป็นผู้เชี่ยวชาญได้ทุกคน”
